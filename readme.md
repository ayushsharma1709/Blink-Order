# Blink Order

A simple and addictive memory game inspired by the classic Simon Says challenge.

Test your memory by repeating increasingly difficult color patterns.  
The game gets faster and harder as your level increases.

## Features

- Interactive gameplay
- Random sequence generation
- Increasing difficulty
- Score/level tracking
- Responsive UI

## Tech Stack

- HTML
- CSS
- JavaScript

## How to Play

1. Press any key to start the game
2. Watch the color sequence carefully
3. Repeat the same pattern
4. Survive as long as possible
